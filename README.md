# TRN 同人女测试 (SBTI Remake)

> MBTI算什么？来测测你到底是哪种同人女。

一个基于开源娱乐性人格测试项目（原 SBTI 测试）二次开发的**同人女主题专属测试**。本测试将原始的 15 维度人格模型重新解构为同人女的 5 大生态模型，帮助你精准定位自己在赛博同人圈的精神状态。

## 在线体验

👉 [点击开始测试](https://chenyu1025.github.io/TRNTI/)

## 特性

- 🧠 **14种同人女人格** — 12种标准类型（如“十年寡妇”、“赛博战狼”） + 2种隐藏/兜底类型（如“赛博抖M”）
- 📊 **15个同人专属维度** — 搞饭、赛博社交、战斗防御、现生边界、磕糖精神五大模型
- 🎯 **曼哈顿距离匹配** — 基于15维向量的科学匹配算法，精准计算你的同人女纯度
- 🔪 **隐藏彩蛋** — 赛博抖M人格触发机制（当你选择去逆家粮里找刺激时...）
- 📱 **移动端优先** — 响应式设计，手机体验友好
- 🔧 **易于定制** — 数据与代码分离，改 JSON 即可创建你自己的测试

## 五大同人女模型解析

1. **搞饭模型（C）**：食谱洁癖度、产出驱动力、搞谷消费观
2. **赛博社交模型（S）**：亲友依恋度、同好社交欲、饭圈融合度
3. **战斗防御模型（B）**：广场战斗力、正主祛魅度、雷点过敏症
4. **现生边界模型（R）**：次壁坚固度、赛博驻扎率、情绪发疯度
5. **磕糖精神模型（M）**：抠糖敏锐度、视界代入感、坑底焊死度

## 评分算法

1. **求和**：每维度2题分值相加（范围2-6）
2. **分级**：≤3 → L（低），4 → M（中），≥5 → H（高）
3. **向量化**：L=1, M=2, H=3，生成15维数值向量
4. **匹配**：计算用户向量与每种同人女类型的曼哈顿距离
5. **排名**：距离升序 → 精准命中降序 → 相似度降序
6. **特殊覆盖**：赛博抖M彩蛋 > 正常匹配 > 混沌邪恶人兜底（<60%）
